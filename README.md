# FitZone Studio - Fitness Studio Website

A dynamic fitness studio website built with React and Tailwind CSS.

## Features

- Class schedules and booking
- Trainer profiles
- Membership signup
- Responsive design
- Interactive class booking modal

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Deploy to Vercel

1. Install Vercel CLI: `npm i -g vercel`
2. Run: `vercel`
3. Follow the prompts to deploy

## Tech Stack

- React 18
- Vite
- Tailwind CSS
